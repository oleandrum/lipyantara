About the schemes

IAST is the standard academic romanization used in print (diacritics like ā, ś, ṛ). Harvard-Kyoto and ITRANS are ASCII-only schemes for typing without special characters — Harvard-Kyoto uses capitalization (saMskRtam), ITRANS reads closer to English spelling conventions. Velthuis is another ASCII scheme, built around punctuation marks (sa.msk.rtam) instead of case. Auto-detect guesses the input scheme automatically, if the source is left unset.
